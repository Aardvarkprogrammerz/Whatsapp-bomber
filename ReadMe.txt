Note : Python is a prerequisite for this project. If you dont have python installed kindly install it or contact Aardvark_Programmerz on instagram for executable file.
1) Python install for windows 7 : python 3.7 or earlier
2) Python install for windows 10 : any python version (recommended : 3.9.1)
3) Python install for debian-based linux : It comes pre-installed!!!

Steps to be followed :
1) Install the required modules
	- pyautogui : pip install pyautogui
	- keyboard : pip install keyboard
* In case of any error, contact us on instagram
2) Create a file named main.py.
3) Paste your code into that file.
4) Replace the value '10' with the no. of messages on line 3. (Recommended : <1000)
5) Run the python file.
6) Place your cursor on the messagebox of the person you want to bomb.
7) Press 'Ctrl + Shift + z'.
8) See the magic.
